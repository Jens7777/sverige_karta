# sverige_karta

Statisk GitHub Pages-version av verktyget finns på:

https://jens7777.github.io/sverige_karta/

Publicering:

1. Pusha ändringarna till `main`.
2. Gå till repot på GitHub.
3. Öppna `Settings` > `Pages`.
4. Välj `Source: GitHub Actions` om GitHub ber dig välja publiceringskälla.
5. Vänta tills workflowen `Deploy GitHub Pages` har kört klart.

Lokal körning:

- Kör `starta_lokal_karta.bat` för att öppna verktyget via `http://localhost:8000/`.
